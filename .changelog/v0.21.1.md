# Release v0.21.1 - Changelog

Released: 2026-02-24

## Overview

Theme support with 5 dark themes.

## Features

### Theme switcher
- **5 dark themes** — Midnight (default, current look), Phosphor (terminal green), Ember (warm orange), Arctic (ice blue), Synthwave (neon purple)
- Palette icon in sidebar footer and mobile header opens a theme popover
- Selection persists across sessions via localStorage
- All existing `port-*` Tailwind classes respond automatically via CSS custom properties — zero component changes needed
- Shell terminal, toast notifications, scrollbars, and focus rings all adapt to the active theme

## Changed

### Safer defaults for new installations
- All 13 CoS self-improvement tasks default to **disabled** — users opt in to the ones they want
- Master `selfImprovementEnabled` switch defaults to off
- Default autonomy level changed from `manager` to `standby`
- Autobiography story prompt defaults to disabled
