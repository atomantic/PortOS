# Release v1.8.x - Changelog

Released: YYYY-MM-DD

## Overview

MeatSpace: Physical health dashboard with death clock countdown, LEV 2045 tracker, and health spreadsheet import.

## Features

### MeatSpace — Health Tracker & Death Clock (Phase 1)
- **New top-level MeatSpace page** (`/meatspace`) with 6 tabs: Overview, Alcohol, Blood & Body, Lifestyle, Nutrition, Import
- **Death Clock Countdown**: Live years/months/weeks/days/hours/minutes/seconds countdown to estimated death date, combining SSA actuarial baseline (78.5y) with genome-adjusted life expectancy from Digital Twin and lifestyle adjustment multipliers (smoking, alcohol, exercise, sleep, diet, stress, BMI)
- **LEV 2045 Tracker**: Longevity Escape Velocity progress visualization — research timeline progress bar, on-track/at-risk status, survival margin to age 66
- **TSV Spreadsheet Import**: Bulk import from 257-column health spreadsheet (3 header rows + 2 summary rows + daily data). Parses nutrition, alcohol (individual beverages with ABV/serving size), body composition, blood tests, DNA methylation (Elysium Index), and eye prescriptions. Idempotent — re-importing replaces all data
- **Automatic sex detection** from genome Y-chromosome SNP count (>1000 = male)
- **Lifestyle adjustment formula**: Additive years for 7 factors — smoking (0 to -10), alcohol (+0.5 to -5), exercise (+2 to -2), sleep (+1 to -1.5), diet (+2 to -3), stress (+1 to -2), BMI (+0.5 to -3)
- **Sidebar navigation**: MeatSpace collapsible section with alphabetically ordered children (Alcohol, Blood & Body, Import, Lifestyle, Nutrition, Overview)
- **Data model**: Sparse daily-log.json (nutrition, alcohol, body, mercury per day), blood-tests.json (with reference ranges), epigenetic-tests.json, eyes.json — all created on import
- New API endpoints: `GET /api/meatspace`, `GET/PUT /api/meatspace/config`, `PUT /api/meatspace/lifestyle`, `GET /api/meatspace/death-clock`, `GET /api/meatspace/lev`, `POST /api/meatspace/import/tsv`
- 24 new tests covering death clock math (lifestyle adjustments, SSA baseline fallback, negative adjustments, expired dates), LEV computation, and TSV import (date parsing, sorting, null handling, body composition, file output)
- Installed `recharts` charting library for future chart components (Phase 2+)
